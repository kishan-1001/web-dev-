## About
This repository serves as a personal learning hub for web development. It includes step-by-step tutorials, coding exercises, and mini-projects covering frontend and backend technologies. Each folder contains starter files, solutions, and goal images to guide the learning process.

## Getting Started
### Prerequisites
- Node.js (for backend projects)
- A code editor (VS Code recommended)
- Git for version control

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/web-dev.git
   cd web-dev
   ```

2. For Node.js projects, install dependencies:
   ```bash
   cd [project-folder]
   npm install
   ```

## How to Use
1. Navigate to the desired tutorial or project folder
2. Open `index.html` in your browser for frontend projects
3. For backend projects, run `node index.js` or `npm start`
4. Compare your code with the provided solutions
5. Use goal images as reference for expected output

### Running Specific Projects
- **Express Server Projects**: `cd [folder] && npm start`
- **API Projects**: Use Postman to test endpoints
- **Frontend Projects**: Open HTML files directly in browser
